# OOP Game Show App

word guessing game: "Phrase Hunter."<br>
Using JavaScript and OOP (Object-Oriented Programming) to select a random, hidden phrase.<br>
A player tries to guess the phrase by selecting individual letters from an onscreen keyboard.<br>
